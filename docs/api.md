# API 仕様書

コトコトバ バックエンドの REST API エンドポイント仕様です。

**Base URL**

- ローカル: `http://localhost:8000`
- 本番: Render にデプロイ（URL は環境による）

**Swagger UI**: `{Base URL}/docs`

---

## エンドポイント一覧

| メソッド | パス             | 説明                         |
| -------- | ---------------- | ---------------------------- |
| GET      | `/`              | ヘルスチェック               |
| GET      | `/vector/health` | Vector 機能の死活確認        |
| POST     | `/vector/init`   | ゲーム開始ワードの取得       |
| POST     | `/vector/calc`   | ベクトル混合演算             |

## `POST /vector/init`

ゲーム開始時にスタートワードをランダムで返します。

### リクエスト

```json
{
  "goal_word": "100億"
}
```

| フィールド  | 型     | 必須 | 説明                       |
| ----------- | ------ | ---- | -------------------------- |
| `goal_word` | string | ✅   | ゴールとなる単語           |

### レスポンス

```json
{
  "start_word": "宇宙",
  "rank": 12345,
  "hint_words": ["銀河", "天体", "惑星", "太陽系", "恒星", "星雲"],
  "description": "宇宙とは、…"
}
```

| フィールド    | 型       | 説明                                     |
| ------------- | -------- | ---------------------------------------- |
| `start_word`  | string   | ランダムに選ばれたスタート単語           |
| `rank`        | integer  | `goal_word` から見た類似度順位           |
| `hint_words`  | string[] | 次に試すべきおすすめ単語（最大 6 件）    |
| `description` | string   | Wikipedia から取得した単語の概要         |

### スタート候補ワード

以下の単語からランダムに選択されます:

> 投資、学校、宇宙、魔法、侍、コンピュータ、恋愛、筋肉、インターネット、時間、料理、人工知能、地球、歴史、音楽、スポーツ

## `POST /vector/calc`

現在のワードとユーザー入力ワードをベクトル混合し、結果ワード・ランク・ヒントを返します。

### リクエスト

```json
{
  "goal_word": "100億",
  "current_word": "宇宙",
  "input_word": "投資",
  "mix_ratio": 0.5
}
```

| フィールド     | 型     | 必須 | 説明                                |
| -------------- | ------ | ---- | ----------------------------------- |
| `goal_word`    | string | ✅   | ゴールとなる単語                    |
| `current_word` | string | ✅   | 現在のワード                        |
| `input_word`   | string | ✅   | ユーザーが入力した混合用ワード      |
| `mix_ratio`    | float  | ✅   | 混合比率（0.0〜1.0）               |

#### `mix_ratio` について

ベクトル線形補間の比率を指定します:

```
v_new = (1 - mix_ratio) × v_current + mix_ratio × v_input
```

- `0.0`: current_word のベクトルをそのまま使用
- `0.5`: 両方のベクトルを均等に混合
- `1.0`: input_word のベクトルをそのまま使用

### レスポンス（成功: 200）

```json
{
  "new_word": "資産",
  "rank": 5678,
  "hint_words": ["経済", "銀行", "金融", "市場", "企業", "資本"],
  "description": "資産とは、…"
}
```

| フィールド    | 型       | 説明                                     |
| ------------- | -------- | ---------------------------------------- |
| `new_word`    | string   | 混合ベクトルに最も近い単語               |
| `rank`        | integer  | `goal_word` から見た類似度順位           |
| `hint_words`  | string[] | 次に試すべきおすすめ単語（最大 6 件）    |
| `description` | string   | Wikipedia から取得した単語の概要         |

### エラーレスポンス（422）

入力単語が word2vec の辞書に存在しない場合:

```json
{
  "detail": "「ほげほげ」は辞書にありません。"
}
```

---

## ヒントワード生成ロジック

ヒントワードは、現在の単語をベースに `goal_word` 方向へ少し寄せたベクトルから近傍単語を抽出して生成されます。

```
v_hint = (1 - hint_ratio) × v_base + hint_ratio × v_goal
```

- `hint_ratio` はデフォルト `0.2`（やや控えめなヒント）
- 現在の単語・入力単語・ゴール単語は候補から除外
- 最大 6 件を返却

---

## ランク判定とクリア条件

- ランクは `goal_word` の近傍順位で計算（`gensim.KeyedVectors.rank()` を使用）
- **ランク 10 位以内でクリア**
- ランクに応じて UI 演出が変化:

| ランク         | 演出                     |
| -------------- | ------------------------ |
| 10,000位以下   | 白黒のシンプルなデザイン |
| 1,000〜9,999位 | 淡いカラーが入り始める   |
| 11〜999位      | 宇宙空間・星のアニメーション |
| 1〜10位        | 黄金色の豪華な演出       |
