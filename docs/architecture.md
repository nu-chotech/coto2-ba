## データフロー

### ゲーム開始フロー

```
User → [シャッフルボタン] → Frontend → POST /vector/init → Backend
                                                              │
                                              VectorEngine.get_start_word()
                                              ├── ランダムにスタート単語を選択
                                              ├── goal_word との rank を計算
                                              ├── ヒントワードを生成
                                              └── Wikipedia から説明を取得
                                                              │
User ← [スタート単語 + ランク + ヒント + 説明] ← Frontend ← Response
```

### ベクトル混合フロー

```
User → [単語入力 + mix_ratio 設定] → Frontend → POST /vector/calc → Backend
                                                                      │
                                                      VectorEngine.calc()
                                                      ├── 入力単語の検証
                                                      ├── ベクトル線形補間:
                                                      │   v_new = (1-ratio)*v_current + ratio*v_input
                                                      ├── 最近傍単語の検索
                                                      ├── goal_word との rank を計算
                                                      ├── ヒントワードを生成
                                                      └── Wikipedia から説明を取得
                                                                      │
User ← [新単語 + ランク + ヒント + 説明] ← Frontend ← Response
```

## ベクトル演算の仕組み

コトコトバのコアとなるベクトル演算は、word2vec の分散表現を利用しています。

1. **単語のベクトル化**: 各単語は 200 次元のベクトルとして表現される
2. **線形補間による混合**: `v_new = (1 - mix_ratio) × v_current + mix_ratio × v_input`
3. **最近傍検索**: 混合ベクトルに最も近い既存の単語を結果として返す
4. **ランク計算**: ゴール単語（"100億"）との類似度順位を算出
